# Electron + Svelte + Tailwind

A quick ElectronJS App starter template using:
 * Svelte
 * Tailwind (JIT)
 * Electron Builder

## Install
```
git clone https://github.com/issmeftah/svelte-electron-tailwind

cd svelte-electron-tailwind/

npm install
```

## Dev
`npm run dev`

## Build
`npm run electron:build:mac`
